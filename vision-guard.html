Sistem Keamanan Face Recognition Terintegrasi (Hardware & Software)

Local Screen Recording Demo

Menampilkan integrasi antara pemicu hardware dan respons dashboard web secara real-time.

Overview

Proyek ini dikembangkan untuk mengotomatisasi sistem keamanan lingkungan dengan memanfaatkan AI. Sistem bekerja saat sensor PIR mendeteksi gerakan, memicu ESP32-Cam untuk menangkap gambar, dan mengirimkannya ke server Django untuk diproses menggunakan OpenCV.

Hardware Stack

  • • ESP32-Microcontroller
  • • PIR Motion Sensor
  • • Hikvision CCTV Integration
  • • Power Management System

Software Stack

  • • Python (OpenCV / AI Core)
  • • Django Framework (Backend)
  • • MySQL (Database)
  • • Tailwind CSS (Dashboard UI)
